# Lab9
Methods Used
The study began with data preprocessing, followed by splitting the dataset into training and testing sets. Two classification models were developed: a Decision Tree classifier and a Random Forest classifier. Model performance was evaluated using multiple metrics, including the Confusion Matrix, Classification Report, and Accuracy Score.

Results
The Random Forest model generally outperformed the Decision Tree model. It showed strong performance in predicting class 0, while its performance for class 1 was weaker, likely due to class imbalance in the dataset.

Conclusion
Both Decision Tree and Random Forest algorithms are effective for classification tasks. However, Random Forest typically provides higher accuracy and more stable results compared to a single Decision Tree. Despite this, imbalanced data can negatively impact performance, highlighting the need for appropriate data balancing techniques.
